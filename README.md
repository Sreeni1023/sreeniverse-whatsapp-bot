# 🚀 SreeniVerse WhatsApp Business Bot

> Production-ready WhatsApp automation bot for SreeniVerse Technologies — handles service inquiries, pricing, bookings 24/7.

**Built by SreeniVerse Technologies, Hyderabad**

## 🛠️ Tech Stack
- WhatsApp: Baileys v7 (Multi-Device)
- Runtime: Node.js v20 + PM2
- Server: Contabo VPS (Ubuntu 24)

## ✨ Features
- 24/7 automated replies
- Full services & pricing menu
- Lead capture system
- Hi loop protection (max 3 welcomes)
- Night mode (10PM-7AM throttle)
- Daily message limit protection
- Blacklist / unsubscribe support
